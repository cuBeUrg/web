Portability

By design, C# is the programming language that most directly reflects the underlying Common Language Infrastructure (CLI).

Typing

C# supports strongly, implicitly typed variable declarations with the keyword var, and implicitly typed arrays with the keyword new[] followed by a collection initializer.

Property

C# supports classes with properties. The properties can be simple accessor functions with a backing field, or implement getter and setter functions.

Namespace

A C# namespace provides the same level of code isolation as a Java package or a C++ namespace, with very similar rules and features to a package

Facts

C# language is good for developing games. It is also used by Unity (the majority leader in commercial game engines) to develop games.
